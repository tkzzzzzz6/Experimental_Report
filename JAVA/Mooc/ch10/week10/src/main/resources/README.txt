在BMI计算器中添加图片的步骤：

1. 获取适合的图片（推荐使用BMI相关的健康图片或图表）
2. 将图片重命名为"background.png"（或修改Main.java中的IMAGE_PATH常量）
3. 将图片文件放置在此目录（src/main/resources/）中

程序会将图片显示在应用界面的右下角。图片会自动调整为合适的尺寸，并显示在所有界面元素的上层。

如果程序启动后无法显示图片，请检查：
- 图片文件是否存在于正确的路径
- 图片文件格式是否正确（推荐使用PNG格式）
- 图片文件是否可以被Java程序正常读取

如果您想调整图片的位置或大小，可以修改Main.java文件中的imagePanel相关设置。
